# GitPeek

GitPeek is a simple web app that lets you view any GitHub user's profile — including their avatar, name, followers, and public repositories — using the GitHub API.

## Features
- Search any GitHub username  
- Displays profile picture, name, followers, and repo count  
- Built using HTML, CSS, and JavaScript (XMLHttpRequest)

## Live Demo
[View GitPeek Live](https://omkar-bhalekar.github.io/GitPeek/)

## How to Use
1. Enter a GitHub username in the input box  
2. Click **Search**  
3. Instantly see the user’s details fetched via the GitHub API  

## Technologies Used
- HTML  
- CSS  
- JavaScript  
- GitHub API
